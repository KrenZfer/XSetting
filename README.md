### Пакет добавляет в Orchid Platform редактирование настроек

## Installation

Install package

```php
composer require orchids/xsetting
```

update migration
```php
php artisan migrate
```

[Инструкция по созданию плагина](https://github.com/orchidcommunity/XSetting/blob/master/docs/new_package_ru.md)
